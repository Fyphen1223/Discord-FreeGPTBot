# Run local
```
git clone https://github.com/Fyphen1223/Discord-FreeGPTBot
```

```
cd Discord-FreeGPTBot
```

```
npm i
```

And then set token to env.token, and applicationId to env.applicationId.

To register slash commands, 
```
node slash.js
```

To run, 
```
node index.js
```
